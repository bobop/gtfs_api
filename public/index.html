Documentation

Here’s how to get your data:

  1. Agencies

    /agencies returns a list of all agencies

    Params:

    ?name=fing returns results from search in agency_name

  2. Routes

    /routes returns a list of 20 routes.

    Params:

    ?limit=50 returns specified number of routes instead of default 20.

    ?name=wythen returns results from search in route_short_name and route_long_name ignoring the case of the characters.

    ?agency_id=GMS returns routes from agency with this agency_id.

    ?include_trips=true returns trips for each route.

    ?include_agency=true returns the agency for each route.

  3. Calendars

    /calendars returns a list of 200 calendar records.

    Params:

    ?limit=500 returns specified number of calendar records instead of default 200.

  4. Calendar Dates

    /calendar_dates returns a list of 200 calendar date records.

    Params:

    ?limit=500 returns specified number of calendar date records instead of default 200.

  5. Stops

    /stops returns a list of 20 stops.

    Params:

    ?limit=50 returns specified number of stops instead of default 20.

    ?stop_id=0600MA0171 returns single stop.

    ?include_stop_times=true returns stop times for each stop.

    ?include_stop_times=true&from=now only returns stop times for each stop with departure_time after right now.

    ?include_stop_times=true&stop_time_limit=5 returns specified number of stop times for each stop (needs ?include_stop_times=true).

  6. Stop Times

    /stops_times returns a list of 200 stop times.

    Params:

    ?limit=500 returns specified number of stops instead of default 200.

  7. Trips

    /trips returns a list of 20 trips.

    Params:

    ?limit=50 returns specified number of trips instead of default 20.

    ?include_calendar=true returns calendar records for each trip.
